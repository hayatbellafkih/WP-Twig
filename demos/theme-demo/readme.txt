 le dossier "oblique" contient le thème démo
 il s'agit du thème https://wordpress.org/themes/oblique/ , il est réecit avec le plugin WP-Twig
 a savoir :
 1- à tout moment, il est possible de basculer vers le meme thème 
 avec PHP, à condition de ne pas supprimer les fichiers PHP
 2- aprés le basculement, il ne faut pas laisser les traces 
  du plugin WP-Twig dans le fichiers functions.php
 3- le présent thème est la version Twig sans passage des
   données via le fichier functions.php, pour plus des exemples à 
   propos le passage des données, veuillez lire readme.txt présent dans 
   le répertoire du plugin
